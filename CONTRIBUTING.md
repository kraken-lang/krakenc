# Contributing to Kraken Compiler

Thank you for your interest in contributing to the Kraken Compiler.

## Current Status

This repository is currently a placeholder. The compiler is being developed in the main [Kraken monorepo](https://github.com/kraken-lang/kraken) under the `compiler/` directory.

## How to Contribute

Until this repository is populated with the compiler source code, please direct all contributions to the main Kraken repository:

- **Repository**: https://github.com/kraken-lang/kraken
- **Issues**: https://github.com/kraken-lang/kraken/issues
- **Discussions**: https://github.com/kraken-lang/kraken/discussions

## Future Contribution Guidelines

Once the compiler is migrated to this repository, we will provide detailed contribution guidelines covering:

- Code style and formatting
- Testing requirements
- Pull request process
- Development environment setup
- Architecture documentation

## Code of Conduct

All contributors are expected to follow the Kraken project's Code of Conduct, which promotes a welcoming and inclusive environment for everyone.

## Questions

For questions about the Kraken compiler or language, please use the discussions forum in the main repository.
